# NoteX

Offline-first knowledge and notes web app.

Core stack:

- React
- TypeScript
- Vite
- TailwindCSS
- React Router
- IndexedDB
- PWA
- Vercel deployment

V1 scope:

- Local-first
- Offline-first
- No backend
- No cloud sync
- Manual export/import

Critical rule:
The UI must match the provided reference images exactly in:

- layout structure
- spacing
- proportions
- sidebar width
- card sizes
- colors
- dark theme surfaces
- borders
- tags/chips
- note detail layout
- dashboard layout
- profile page layout

Do not redesign the product.
Recreate the provided UI faithfully.
