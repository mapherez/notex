# Pages

## Dashboard

Includes:

- Search
- Stat shortcuts for notes, favorites, collections, tags, and trash
- Quick pins
- Recent notes
- Quick capture
- Popular tags
- Recent activity

## Notes

The primary `/notes` experience shows block-based notes with a title, subtitle, thumbnail, tags, table of contents, Tiptap content blocks, inline images, file attachments, and configurable side panels.

## Tags

Includes tag creation, renaming, color updates, deletion, and note filtering by tag.

## Collections

Includes collection creation, renaming, color updates, deletion, and note filtering by collection.

## Profile

Includes:

- Local account identity
- Basic statistics
- Preferences
- SQLite database path
- Local `.notex` package export/import and local data management

## Legal

`/privacy` and `/terms` render the app privacy and terms content.
