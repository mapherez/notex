# Global Layout

## Left Sidebar

Always visible on desktop.
Drawer on mobile.

Contains:

- Logo (NoteX)
- New Note CTA
- Home
- Notes
- Favorites
- Recent
- Trash
- Collections
- Storage block
- User mini profile footer

## Top Bar

Contains:

- Search bar
- Theme toggle
- Notifications
- User avatar

## Design Rules

- Premium dark theme default
- Rounded corners
- Subtle borders
- Layered dark surfaces
- Minimal gradients
- Purple accent color
- Controlled spacing
- No generic SaaS dashboard look
